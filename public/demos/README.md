# Demo Songs

Place Suno-generated MP3s here following the naming pattern from
`src/lib/demoSongs.ts`:

- `rock_pop/rock-1.mp3` … `rock-4.mp3`
- `lofi/lofi-1.mp3` … `lofi-3.mp3`
- `electronic/edm-1.mp3` … `edm-4.mp3`
- `classical/cls-1.mp3` … `cls-3.mp3`
- `podcast/pod-1.mp3` … `pod-3.mp3`

When a file is missing, the UI surfaces a friendly "coming soon" alert
instead of failing.
