# AuSpec — Project Overview

**Tagline:** Turn Audio Into Living Visuals

## What is AuSpec?
AuSpec is a web-based audio spectrum visualizer. Users upload audio files and generate reactive visual animations such as waveforms, circular spectrums, particle systems, and customizable music visuals.

## Vision
"Canva for music visuals."

## Core User Flow
1. Upload audio file (mp3, wav, m4a, flac)
2. Audio plays in browser
3. Live frequency data is analyzed
4. Reactive visualization renders on canvas
5. User customizes colors, forms, motion, effects
6. User uploads cover art / logo
7. User exports visual as video
