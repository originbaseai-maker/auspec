# Phase 9 — Social Format System

## Goal
Canvas supports multiple aspect ratios for different social platforms.

## Status
- [ ] Not Started

## Supported Ratios
| Format | Ratio | Platform |
|--------|-------|----------|
| Landscape | 16:9 | YouTube |
| Portrait | 9:16 | TikTok / Reels |
| Square | 1:1 | Instagram |
| Portrait | 4:5 | Instagram Feed |
| Ultrawide | 21:9 | Cinematic |

## Behavior
- Canvas resizes dynamically on format change
- All visual elements scale proportionally
