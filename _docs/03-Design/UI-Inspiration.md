# UI Inspiration

## Primary References
- [Linear.app](https://linear.app) — dark UI, clean sidebar, minimal controls
- [Framer](https://framer.com) — canvas-centric layout, premium feel
- [Vercel](https://vercel.com) — dark theme, typography, spacing
- [Canva](https://canva.com) — preset system, side panels
- [Figma](https://figma.com) — studio layout, tool panels

## Studio Layout Reference

- Center: main canvas (full height)
